# Lounge Tone Application

When writing for the Lounge, apply the following adjustments:

1. **Simplicity over cleverness:** Remove complex industry jargon. If you must use a term like "Agentic RAG", define it gently in a single sentence.
2. **Empathetic pacing:** Use frequent line breaks, short paragraphs, and bullet points. It must be Skimmable.
3. **Inject Humor sparingly:** Use "I tried this so you don't have to" framing and light, relatable frustration triggers (e.g., "We've all stared at a broken YAML file at 2 AM...").
4. **Emoji usage:** Keep it light. Maximum of 4-5 total emojis per 800 words. No over-the-top reactions.
