# Scout Daily Topics

1. **Scan Platforms:** Use web search to discover trending conversations on X, Reddit (r/ClaudeAI, r/MachineLearning, r/LangChain), and official anthropic development blogs.
2. **Filter Topics:** Look strictly for subjects related to Claude Code, Agentic workflows, OpenClaw framework derivatives (ZeroClaw, AutoClaw), or CLI productivity.
3. **Compile Summary:** Generate a list of 5 key topics.
4. For each topic, draft a 1-sentence "Why it matters to learners" summary.
5. Output the final summary into `content/lounge_drafts/daily_scout_report.md`.
