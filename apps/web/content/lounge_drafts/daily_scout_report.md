1. **The MCP Explosion in Database Auth:** Community is buzzing about Model Context Protocol finally replacing clunky middle-layer APIs for direct local Postgres -> Claude Code connections.

2. **Context Window "Amnesia":** Users on r/ClaudeAI complaining about over-stuffed context windows forgetting initial instructions halfway through Swarm executions.
   *Why it matters to learners:* Reinforces our pedagogy around specific, modular NanoClaw agents vs massive orchestration.
3. **The 'Stop, Undo, Recover' Methodology:** Becoming a standard paradigm over "try until it works", heavily discussed alongside ZeroClaw's simplicity.
   *Why it matters to learners:* Shifts the mindset from fear of failure to absolute control over the agent loop.
4. **Tool Use vs Multi-Agent Debate:** Heated discussions on when "one smart agent with 5 tools" beats "5 agents talking to each other".
   *Why it matters to learners:* Direct impact on how they architects their Capstone projects.
5. **ZeroClaw Onboarding Funnels:** Widespread praise for ZeroClaw as the absolute lowest barrier to entry for orchestrating Claude models locally.
   *Why it matters to learners:* Confirms our primary entry-level technical stack choice.
