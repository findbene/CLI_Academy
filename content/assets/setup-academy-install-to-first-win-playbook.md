# Setup Academy Install-to-First-Win Playbook

Use this beside Week 1 of the fast path.

## 1. Prepare the machine

- Confirm which shell you are using.
- Confirm where your workspace should live.
- Confirm you know the stop, undo, and recover routine before installing anything.

## 2. Install and verify

- Install the required runtime or CLI from the recommended source.
- Open a fresh shell after install.
- Run the version check and save the output in your notes folder.

## 3. Authenticate carefully

- Sign in with the correct account.
- Keep auth and environment variable notes in `notes/`, not scattered screenshots.
- If auth fails, use the recovery checklist before repeating the install.

## 4. First useful task

- Start with a bounded read or tiny edit task.
- Review the diff or output before trusting it.
- Capture one “what worked” note and one “what confused me” note.

## 5. Graduation check

- The tool opens cleanly.
- The first command succeeds.
- You know where logs, notes, screenshots, and recovery steps live.
- You can explain the first useful win in plain English.
