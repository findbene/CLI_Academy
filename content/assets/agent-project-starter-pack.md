# Agent Project Starter Pack

This starter pack is for learners moving beyond setup into structured agentic project work.

## Included Ideas

- A stronger `CLAUDE.md` baseline
- A simple `MEMORY.md` pattern for project memory
- Suggested folders for prompts, outputs, notes, and automations
- Guardrails for safe iteration

## Suggested Starter Structure

```text
project-root/
  CLAUDE.md
  MEMORY.md
  prompts/
  docs/
  outputs/
  scripts/
```

## Recommended Rules

- Keep reusable prompts in one place.
- Store generated outputs separately from source material.
- Update `MEMORY.md` when the project learns something worth preserving.
- Do not let agents invent folder structure differently on every run.

## Best For

- MCP experiments
- automation projects
- internal tools
- founder/operator workflows
