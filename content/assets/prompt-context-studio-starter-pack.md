# Prompt And Context Studio Starter Pack

Use this pack when a rough request needs to become a reusable workflow.

## Prompt Anatomy

Every strong prompt should make these explicit:

- role
- task
- inputs
- constraints
- output format
- review criteria

## First Rewrite Pass

Turn vague requests into:

- what you want done
- what good looks like
- what to avoid
- what the model should ask before proceeding

## Context Packet Checklist

- repo or folder boundary
- source files or trusted docs
- operating system or shell assumptions
- allowed tools
- forbidden actions
- stop conditions

## Prompt Doctor Questions

- Is the task too broad?
- Are the inputs complete?
- Is the output shape explicit?
- Is the context overloaded?
- Is the model being asked to take unsafe actions?

## Save-For-Later Prompts

Keep templates for:

- explain before edit
- rewrite this into a session brief
- compare two options and recommend one
- retrieve and ground before answering
