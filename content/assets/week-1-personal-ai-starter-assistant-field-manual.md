# Week 1 Personal AI Starter Assistant Field Manual

## Mission

Turn a clean local setup into one small assistant you can use again tomorrow.

## The smallest acceptable version

- A safe workspace folder exists.
- Claude Code runs correctly.
- You completed one useful builder action.
- You stored your notes, screenshots, and recovery steps in predictable places.

## Daily loop

1. Open the workspace.
2. Check the task scope.
3. Run the smallest useful command or prompt.
4. Review the output.
5. Save one note about what to repeat next time.

## Red flags

- You are copying commands you do not understand into a protected directory.
- You are retrying auth or installs without checking logs or PATH first.
- You are asking the tool to do several risky things at once.

## End-of-week proof

- Screenshot or text record of a clean version check
- One successful starter task
- One written recovery path for the most likely failure
- One sentence that explains what your starter assistant now does
