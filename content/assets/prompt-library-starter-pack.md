# Prompt Library Starter Pack

## Claude Code setup prompt

```
Explain whether my machine is ready for Claude Code.
Check the shell, runtime version, PATH visibility, and the safest next step.
Keep the explanation beginner-friendly and give me one verification command.
```

## Repo exploration prompt

```
Before editing anything, explain this repo like I am new to it.
Show me the entry point, the most important files, and one safe tiny improvement I could make first.
```

## Cowork briefing prompt

```
Use the files in this folder to produce a clean briefing pack.
Separate facts, assumptions, missing information, and recommended next actions.
Do not invent details that are not grounded in the source files.
```

## Troubleshooting prompt

```
I hit an error during setup.
Rank the top three likely causes from most likely to least likely.
Give me the safest verification step for each before suggesting a fix.
```
