# Capstone Showcase Submission Kit

Use this when packaging the final Autonomous AI Workforce for portfolio proof or the optional community showcase.

## Submission bundle

### README

- problem the workforce solves
- who it is for
- architecture summary
- setup steps
- safety boundaries
- what is still manual versus automated

### Evidence pack

- screenshots or terminal captures
- sample outputs
- evaluation notes
- failure cases and how you handled them

### Demo script

- 2-minute overview
- one safe live run
- one review checkpoint
- one lesson learned

### Architecture diagram prompts

- system boundaries
- planner-worker-reviewer flow
- retrieval layer
- deployment and monitoring layer

## Showcase rule

The goal is not hype. The goal is clear, honest proof that the learner built something real, bounded, and reviewable.
