# Chapter Mini Notes Series

Use this as the standard one-page printable for every CLI Academy chapter.

## Required sections

1. Outcome in plain English
2. Key commands or actions
3. What good looks like
4. Common mistakes
5. Quick verification checklist
6. What unlocks next

## Why this exists

- Beginners should not need to reread a full lesson to restart a lab.
- Setup and troubleshooting chapters need an offline fallback.
- Prompt, runtime, and capstone lessons stay more approachable when the learner has a compact recap.

## Suggested chapter packs

- Setup chapters: install checks, auth checks, doctor commands, recovery branches
- Prompt chapters: anatomy reminders, variable patterns, evaluation questions
- Runtime chapters: boundaries, rollback steps, safe defaults, logs to inspect
- Capstone chapters: artifact list, demo script checks, README evidence list
