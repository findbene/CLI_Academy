## Phase 1: Guided Build (Follow Along)

Let's write the core brain of our NanoClaw agent. Open up your favorite code editor (or just stick around in the terminal) and create a file called `nano_agent.py`.

Here is the exact code you need to start:

```python
import os
import shutil

# Step 1: Tell the agent where to look
DESKTOP_PATH = os.path.expanduser("~/Desktop")
ORGANIZE_FOLDER = os.path.join(DESKTOP_PATH, "AI_Organized_Screenshots")

def init_nano_agent():
    print("🤖 NanoClaw Agent Initialized. Scanning Desktop...")
    
    # Step 2: Ensure our target folder exists
    if not os.path.exists(ORGANIZE_FOLDER):
        os.makedirs(ORGANIZE_FOLDER)
        print(f"[NanoClaw] Created new directory: {ORGANIZE_FOLDER}")

if __name__ == "__main__":
    init_nano_agent()
```

<Hint title="What is os.path.expanduser doing?">
  This is a safe way to let your code find the Desktop path regardless of whether you are on a Mac, Windows, or Linux machine. Always trust the `os` library over hardcoding file paths!
</Hint>

## Phase 2: Hint-Based Checkpoint (Try It Yourself)

Now that the agent knows *where* to look, we need to teach it *how* to move files. 

Your challenge: Write a loop that checks every file on the Desktop. If the file is a `.png` image that starts with the word "Screenshot", move it into the `ORGANIZE_FOLDER`.

**Try to write this loop yourself before checking the answer!**

<Hint title="Need a hint for the loop?">
  Use `for filename in os.listdir(DESKTOP_PATH):` to iterate over everything on the desktop. Use `if filename.endswith(".png"):` to check the file type.
</Hint>

<br/>

<Hint title="Show me the full Agent Logic code!">
```python
    count = 0
    for filename in os.listdir(DESKTOP_PATH):
        if filename.startswith("Screenshot") and filename.endswith(".png"):
            source = os.path.join(DESKTOP_PATH, filename)
            destination = os.path.join(ORGANIZE_FOLDER, filename)
            
            # Agent action taking place!
            shutil.move(source, destination)
            count += 1
            print(f"[NanoClaw] Task Complete: Moved {filename}")
```
</Hint>
