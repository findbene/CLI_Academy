# sandbox-foundations — Archived from Path 11

These 7 lessons were the original content of `11-openclaw-and-claw-runtime-foundations` before the April 2026 OpenClaw restoration.

## Why Archived

Path 11's slug has always referenced OpenClaw but the lessons taught Docker sandbox mechanics (process isolation, venv, permission boundaries). During the April 2026 curriculum restoration, path 11 was rewritten to teach OpenClaw foundations — the subject its slug actually promises.

The sandbox content is not worthless. The concepts overlap with:
- Path 14 (Secure Local Machines and Safe Defaults)
- Path 22 (Dev Environment Hardening)
- Path 25 (Agent Safety in Production)

If a future lesson needs to reference sandbox mechanics for context, these files are the source.
