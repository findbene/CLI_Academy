## What You'll See

By the end of this lesson, you will clearly understand what Claude CoWork is, how it differs from Claude.ai and Claude Code, what kinds of work it is designed for, and what "research preview" actually means for how you should use it.

## Why This Matters

Anthropic now ships three distinct products for different audiences. Using the wrong one for a task means worse results and more frustration. Understanding CoWork's actual design purpose lets you use it in the situations where it genuinely excels — and avoid expecting it to do things it was not built for.

## Core Concepts

### Three Products, Three Audiences

Anthropic's main products serve different people doing different things:

**Claude.ai** — A conversational AI in a browser. It answers questions, writes content, and discusses ideas. It does not access your files. It does not take multi-step actions. Use it when you want a thoughtful conversational partner.

**Claude Code** — A terminal-based AI agent for developers. It reads and writes your actual files, runs commands, and understands your codebase. Use it when you are building software.

**Claude CoWork** — A desktop-oriented AI product for agentic knowledge work. It is built for non-technical professionals who need to accomplish multi-step, outcome-oriented tasks — writing complex documents, synthesizing research, analyzing data — without writing code.

### What CoWork Is Built For

CoWork is designed around *outcomes*, not conversations. Where Claude.ai responds to a question, CoWork executes a task.

The tasks it handles well:
- Writing long-form documents (reports, proposals, SOPs) from a brief
- Multi-step research: finding sources, synthesizing, outlining, drafting
- Data analysis: reading structured files, extracting insights, summarizing
- Document editing: improving, restructuring, or expanding existing materials
- Knowledge work that would take a human professional hours

Think of CoWork as the agentic version of having a very competent assistant — one who can do significant desk work, not just answer questions.

### What CoWork Is Not For

CoWork is not a replacement for Claude Code or Claude.ai. It has different strengths:

- **Not for coding.** If you need to write, run, or debug code, use Claude Code.
- **Not for quick questions.** If you want to ask something and get a fast answer, use Claude.ai.
- **Not for system administration.** CoWork does not run shell commands or manage servers.
- **Not yet for everything.** It is in research preview — not all capabilities are complete.

### Research Preview: What It Actually Means

"Research preview" means Anthropic is actively gathering feedback and iterating on the product. In practice:

- Features may change or disappear between versions
- Some capabilities are incomplete or inconsistent
- Anthropic is still learning which use cases CoWork serves best
- You may encounter rough edges, unexpected behaviors, or tasks it handles poorly

This does not mean CoWork is not useful — it means you should approach it as a capable but not fully polished tool. Check outputs carefully. Give feedback when things go wrong.

<TipBlock>Treat CoWork like a talented new hire in their first month. Highly capable, but you need to review their work before sending it out, and you need to give clear instructions rather than vague ones.</TipBlock>

### The Agentic Difference

What makes CoWork different from a chatbot is the multi-step execution. When you give CoWork a task like "write a competitive analysis of three SaaS companies," it does not respond with a single output. It:

1. Breaks the task into steps (research each company, identify comparison dimensions, structure the analysis, write each section)
2. Works through those steps in sequence
3. Produces a polished output rather than a running dialogue

This is what "agentic" means: the system takes initiative on the steps, rather than waiting for you to direct each one.

## Key Takeaways

- Claude CoWork is Anthropic's research-preview product for multi-step, agentic knowledge work.
- It is for non-technical professionals who need to accomplish substantial desk tasks, not for coding or quick questions.
- It differs from Claude.ai (chat) and Claude Code (developer terminal tool) in design and target audience.
- Research preview means useful but not complete — expect rough edges and review all outputs.
- CoWork works through tasks in multi-step sequences, not single-turn responses.
