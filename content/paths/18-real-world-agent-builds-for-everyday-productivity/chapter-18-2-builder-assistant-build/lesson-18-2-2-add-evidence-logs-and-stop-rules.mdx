# Add evidence, logs, and stop rules

## What you will build
A working safety pack: stop-rules.md plus agent-log.md with at least one real recorded run.

## Why this matters
Real-world agent builds are the proof that everything in the curriculum works in practice, not just in isolated exercises. The everyday productivity scenarios in this path mirror the first agentic projects most developers build when they join an organization using Claude Code.

## Before you start
Add a mandatory `agent-log.md` evidence trail and strict stop rules (max iterations, max file count, allowed glob scope) so your assistant cannot run away or silently overreach.


## Walkthrough
### Step 1: Create evidence and rule files

```bash
touch agent-log.md stop-rules.md
```

```powershell
New-Item agent-log.md -ItemType File -Force | Out-Null
New-Item stop-rules.md -ItemType File -Force | Out-Null
```

### Step 2: Define non-negotiable stop rules

```bash
cat > stop-rules.md <<'EOF'
# Stop Rules

1. Max iterations per run: 3
2. Max files processed: 20
3. Allowed paths only: content/drafts/**/*.md, *.md at repo root
4. If command errors twice in a row: stop
5. If user approval is required and not granted: stop
EOF
```

```powershell
@"
# Stop Rules

1. Max iterations per run: 3
2. Max files processed: 20
3. Allowed paths only: content/drafts/**/*.md, *.md at repo root
4. If command errors twice in a row: stop
5. If user approval is required and not granted: stop
"@ | Set-Content stop-rules.md
```

### Step 3: Define evidence log schema
Add this format to `agent-log.md`:

```md
# Agent Evidence Log

| Timestamp | Run ID | Task | Command/Action | Files Touched | Result | Stop Rule Triggered |
| --- | --- | --- | --- | --- | --- | --- |
```

### Step 4: Update CLAUDE.md to enforce logging and stop rules
Add this section:

```md
## Logging and stop enforcement
- Every action must be logged to agent-log.md.
- Before each step, check stop-rules.md.
- If any stop rule is reached, halt immediately and report why.
```

### Step 5: Run one helper task and log it
Prompt Claude Code:

```text
Read CLAUDE.md and stop-rules.md. Execute one helper task within allowed scope only, and append every action to agent-log.md using the defined table format.
```

Then verify:

```bash
cat agent-log.md
```

```powershell
Get-Content agent-log.md
```

### What can go wrong

| Symptom | Likely cause | Fix |
| --- | --- | --- |
| No evidence after run | Logging not mandatory in contract | Add explicit "log every action" requirement |
| Agent processes too many files | Max file rule missing or ignored | Add `max files: 20` and require pre-check each step |
| Agent loops repeatedly | No iteration cap | Enforce `max iterations: 3` |
| Actions outside expected folders | Missing glob/path allowlist | Restrict to explicit patterns in stop-rules.md |
| Cost overrun on long runs | No hard stop on retries | Stop after two consecutive command errors |
