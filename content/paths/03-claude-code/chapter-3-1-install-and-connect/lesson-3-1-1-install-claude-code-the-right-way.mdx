# Install Claude Code the right way

## What you will build
claude --version prints a version number and authentication succeeds.

## Why this matters
Claude Code is the primary interface for agentic development. Understanding how it reads context, executes tools, and takes action is the difference between a developer who directs Claude precisely and one who gets unpredictable results. Every advanced agentic pattern builds on this foundation.

## Before you start
Install Claude Code from npm, authenticate with your Anthropic account, and confirm the tool is ready to use by checking its version. When you finish you will have a working `claude` command available from any terminal window.


## Walkthrough

### Step 1 — Check your Node.js version

Claude Code requires Node.js 18 or newer. Open a terminal and run:

**macOS / Linux**
```bash
node --version
```

**Windows PowerShell**
```powershell
node --version
```

You need `v18.0.0` or higher. If the command is not found or the version is too old, install Node.js from [nodejs.org](https://nodejs.org) (choose the LTS installer) and reopen your terminal before continuing.

### Step 2 — Install Claude Code globally

**macOS / Linux**
```bash
npm install -g @anthropic-ai/claude-code
```

**Windows PowerShell (run as Administrator)**
```powershell
npm install -g @anthropic-ai/claude-code
```

> **Why Administrator on Windows?** Global npm installs write to a system-level folder. Without elevated permissions the install fails with an `EPERM` error. On macOS / Linux you may need to prefix the command with `sudo` if you see a similar permissions error.

Wait for the install to finish. You will see a summary line like `added X packages`.

### Step 3 — Verify the installation

Close your terminal completely and open a fresh one so your shell picks up the new PATH entry. Then run:

**macOS / Linux**
```bash
claude --version
```

**Windows PowerShell**
```powershell
claude --version
```

You should see a version string such as `1.0.16` (your number may differ). If you see it, the binary is installed correctly.

### Step 4 — Authenticate with your Anthropic account

```bash
claude auth login
```

This opens a browser window where you sign in to your Anthropic account and authorize the CLI. After you approve, the terminal prints a success message. If no browser opens, copy the URL shown in the terminal and paste it into any browser on the same machine.

### What can go wrong

| Symptom | Likely cause | Fix |
|---|---|---|
| `npm: command not found` | Node.js is not installed or not on PATH | Install Node.js LTS from nodejs.org and restart your terminal |
| `EACCES` or `EPERM` during install | Missing write permissions to the global npm folder | macOS/Linux: re-run with `sudo`. Windows: reopen PowerShell as Administrator |
| `claude: command not found` after install | Shell PATH has not refreshed | Close the terminal completely and open a new one |
| `claude --version` works but `auth login` shows a network error | Corporate proxy or firewall blocking HTTPS | Set the `HTTPS_PROXY` environment variable to your proxy URL and retry |
| Browser does not open during `auth login` | Headless server, WSL, or SSH session without a GUI | Copy the URL from the terminal and open it in any browser on the same machine |
