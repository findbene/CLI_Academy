## The Guided Path: Your First Run

Follow these exact steps to get your agent running safely. We’ll provide the code, you provide the execution. 

### Step 1: Install ZeroClaw

Open your terminal. Make sure you are in a safe directory (like your Desktop or a dedicated `CLI_Academy_Projects` folder). Run the following command:

<TerminalSimulation
  command="npm i -g @openclaw/zeroclaw"
  output="+ @openclaw/zeroclaw@1.0.0
added 42 packages in 3s"
  status="success"
/>

<Screenshot
  src="/assets/placeholders/install_success.png"
  alt="NPM successfully installing ZeroClaw globally."
  caption="ZeroClaw installs rapidly since it is dependency-light."
/>

### Step 2: Set Up Your API Key Environment

Before ZeroClaw can talk to the LLM (Large Language Model), it needs a way to authenticate. 

*On Windows (PowerShell):*
<CodeBlock language="powershell">
$env:ANTHROPIC_API_KEY="your-api-key-here"
</CodeBlock>

*On macOS / Linux:*
<CodeBlock language="bash">
export ANTHROPIC_API_KEY="your-api-key-here"
</CodeBlock>

<Hint title="API Key Safety Recap">
Never type your API key directly into a script file if you plan to share it! Using environment variables (like we did above) keeps the key safely in your current terminal session and out of your source code.
</Hint>

### Step 3: Write the Agent Script

In your terminal, you can download the starter files directly, or create a brand new file inside your project folder called `hello_agent.js`.

**Download Asset:** [Starter Code Folder](/assets/labs/zero_claw_hello_world/starter/)

Open the `hello_agent.js` file in your editor and paste the following code:

<CodeBlock language="javascript" title="hello_agent.js">
import { ZeroClaw } from '@openclaw/zeroclaw';

async function run() {
  console.log("🤖 Booting ZeroClaw...");

  const agent = new ZeroClaw({
    model: 'claude-3-haiku-20240307',
    system: "You are a friendly, enthusiastic agent welcoming a user to CLI Academy."
  });

  const response = await agent.prompt("Say a short hello and introduce yourself!");
  
  console.log("\nAgent says:");
  console.log(response.text);
}

run().catch(console.error);
</CodeBlock>

## Independent Challenge: Echo & List

Now it is your turn to fly solo. We challenge you to create a new script named `file_agent.js`.

**The Goal:**
1. Start ZeroClaw.
2. Ask the user for input via the standard node `readline` module (or just hardcode a directory path).
3. Have the agent list the files in the directory by utilizing a basic tool integration.

*Wait! Tool integration? ZeroClaw has a `tools` parameter you can pass during instantiation.*

<CodeSandbox
  title="ZeroClaw Tool Snippet"
  code="
  const agent = new ZeroClaw({
    tools: [
      {
        name: 'list_files',
        description: 'Lists files in the current directory.',
        execute: async () => {
          // Can you figure out how to return an array of filenames here?
        }
      }
    ]
  });"
/>

If you get stuck, feel free to download the [Solution Code Folder](/assets/labs/zero_claw_hello_world/solution/) to see how we built it.
