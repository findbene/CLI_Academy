# ZeroClaw Hello World – Your First Agent in Under 10 Minutes

## What you will build
A hello-agent/ project in cli-academy/projects/ containing package.json, hello-agent.mjs, and node_modules/ — with a successful run that printed an AI response and token count.

## Why this matters
Professional developers spend the first minutes of any new project setting up a clean workspace. Starting from an organized structure prevents lost files, confusing paths, and the embarrassment of committing secrets. This habit — workspace first, then work — is the foundation of every reliable agentic workflow.

## Before you start
You will write a small Node.js script that sends a prompt to an AI model and prints the response in your terminal. By the end of this lesson you will have run your first AI agent locally — a real script that talks to a real model and gives you a real answer.


## Walkthrough

### Step 1 — Check that Node.js is installed

This lesson requires Node.js. Run this command to confirm:

**Any OS:**
```bash
node -v
```

You should see a version number like `v18.20.0` or higher. If you see `command not found` or `not recognized`, install Node.js from https://nodejs.org before continuing.

### Step 2 — Create a project folder

**macOS / Linux:**
```bash
cd ~/cli-academy/projects
mkdir hello-agent
cd hello-agent
```

**Windows PowerShell:**
```powershell
Set-Location ~/cli-academy/projects
New-Item -ItemType Directory -Path hello-agent -Force
Set-Location hello-agent
```

### Step 3 — Initialize a Node.js project

```bash
npm init -y
```

This creates a `package.json` file. You should see a JSON block printed in the terminal.

### Step 4 — Install the Anthropic SDK

```bash
npm install @anthropic-ai/sdk
```

You should see output ending with `added X packages`. This gives your script the ability to talk to Claude.

### Step 5 — Set your API key

Your script needs an API key to authenticate with the Anthropic API.

> **Safety warning:** Never paste your API key into a file you will commit to Git or share publicly. Environment variables keep the key in your terminal session only.

**macOS / Linux:**
```bash
export ANTHROPIC_API_KEY="your-api-key-here"
```

**Windows PowerShell:**
```powershell
$env:ANTHROPIC_API_KEY="your-api-key-here"
```

Replace `your-api-key-here` with the real key from https://console.anthropic.com/settings/keys.

To verify it is set:

**macOS / Linux:**
```bash
echo $ANTHROPIC_API_KEY | cut -c1-8
```

**Windows PowerShell:**
```powershell
$env:ANTHROPIC_API_KEY.Substring(0,8)
```

You should see the first 8 characters of your key. Never print the full key.

### Step 6 — Write the agent script

Create a file called `hello-agent.mjs`:

**macOS / Linux:**
```bash
cat <<'SCRIPT' > hello-agent.mjs
import Anthropic from "@anthropic-ai/sdk";

const client = new Anthropic();

async function main() {
  console.log("Starting your first AI agent...\n");

  const message = await client.messages.create({
    model: "claude-sonnet-4-20250514",
    max_tokens: 256,
    messages: [
      {
        role: "user",
        content: "Say hello to a CLI Academy student who just ran their first AI agent script. Keep it to 2-3 sentences."
      }
    ]
  });

  console.log("Agent says:");
  console.log(message.content[0].text);
  console.log("\nTokens used:", message.usage.input_tokens + message.usage.output_tokens);
}

main().catch(console.error);
SCRIPT
```

**Windows PowerShell:**
```powershell
@'
import Anthropic from "@anthropic-ai/sdk";

const client = new Anthropic();

async function main() {
  console.log("Starting your first AI agent...\n");

  const message = await client.messages.create({
    model: "claude-sonnet-4-20250514",
    max_tokens: 256,
    messages: [
      {
        role: "user",
        content: "Say hello to a CLI Academy student who just ran their first AI agent script. Keep it to 2-3 sentences."
      }
    ]
  });

  console.log("Agent says:");
  console.log(message.content[0].text);
  console.log("\nTokens used:", message.usage.input_tokens + message.usage.output_tokens);
}

main().catch(console.error);
'@ | Set-Content hello-agent.mjs
```

### Step 7 — Run it

```bash
node hello-agent.mjs
```

You should see output like:

```
Starting your first AI agent...

Agent says:
Hello and welcome! You just ran your very first AI agent script — that's a
real accomplishment. The terminal might look intimidating now, but you're
already making it work for you.

Tokens used: 97
```

The exact words will be different every time — that is how generative AI works. The important thing is that you see a response and a token count.

**Congratulations. You just ran a real AI agent on your own machine.**

### Step 8 — Experiment (optional)

Open `hello-agent.mjs` in your editor and change the `content` string to ask a different question — for example:

```
"Explain what an API key is in one sentence, for someone who has never programmed before."
```

Save the file and run `node hello-agent.mjs` again. Notice how the response changes completely based on what you ask.

### Step 9 — Commit your work

```bash
cd ~/cli-academy
git add projects/hello-agent/
git commit -m "Add hello-agent: first AI agent script"
```

### What can go wrong

| Problem | Cause | Fix |
|---|---|---|
| `node: command not found` / `not recognized` | Node.js is not installed | Install from https://nodejs.org, restart your terminal, then retry |
| `npm ERR! code ENOENT` | You are not inside the `hello-agent/` folder | Run `cd ~/cli-academy/projects/hello-agent` and retry |
| `Error: Could not load the default credentials` or `401 Unauthorized` | API key is missing or wrong | Re-run the `export` / `$env:` command from Step 5 with your real key. Keys start with `sk-ant-` |
| `ERR_MODULE_NOT_FOUND` for `@anthropic-ai/sdk` | The package was not installed, or you are in the wrong folder | Run `npm install @anthropic-ai/sdk` inside `hello-agent/` |
| `SyntaxError: Cannot use import statement outside a module` | The file extension is `.js` instead of `.mjs` | Rename the file to `hello-agent.mjs` or add `"type": "module"` to `package.json` |
| `429 Too Many Requests` | You hit the API rate limit | Wait 60 seconds and try again |
| Response is different from the example | This is normal for AI — responses vary every time | As long as you see text and a token count, the script worked correctly |
