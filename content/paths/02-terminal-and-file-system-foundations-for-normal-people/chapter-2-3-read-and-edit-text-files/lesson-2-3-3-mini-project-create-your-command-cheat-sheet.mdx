# Mini project: create your command cheat sheet

## Objective
Consolidate all the commands you've learned into your own personal `terminal-cheatsheet.md` that you can reference throughout the rest of your journey.

## Walkthrough

You've learned the fundamental building blocks of terminal and file-system manipulation. You now have complete autonomy over your digital workspace!

You can:
- Look at a map: `pwd` / `cd`
- Scan the room: `ls` / `dir`
- Walk into folders: `cd <folder>`
- Walk out of folders: `cd ..`
- Create folders: `mkdir <folder>`
- Rename and Move: `mv <old> <new>`
- Read files: `cat <file>`
- Duplicate and Backup: `cp <file> <backup>`

**The Project Task:**
It's time to build your first real asset. 

1. Create a brand new folder called `cheatsheets`.
2. Step inside it.
3. Create a file called `terminal-cheatsheet.md`.
4. Open it in a text editor (Notepad, VS Code, or whatever you are comfortable with) and write down the commands you've learned to form a real reference sheet.
5. Save the file.
6. Use `cat terminal-cheatsheet.md` to display your final masterpiece in the terminal!

Paste the output of your `cat` command here so the tutor can verify your beautiful cheat sheet. Congratulations on completing Path 2! You're ready to start coworking with AI.
