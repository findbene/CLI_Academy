# Mini project: create your command cheat sheet

## Objective

Build a real `command-cheat-sheet.md` file by running every command you learned in Path 02, recording the output, and organizing it into a reference you will actually use going forward.

## Walkthrough

### Step 1 — Create the cheat sheet workspace

<StepMeta purpose="Create a dedicated place for your reusable terminal reference instead of mixing it with project files." action="Move to your home directory, create cli-academy/cheatsheets if needed, enter it, and print the path." expectedResult="You are inside a cheatsheets folder under cli-academy and the location is confirmed in the terminal." whyItMatters="A predictable storage location makes the cheat sheet easy to revisit and expand after future lessons." nextStep="Once the workspace exists, create the markdown file and write a simple header into it." askTutor="I'm on Step 1 of Mini project: create your command cheat sheet. Help me verify that I created the cheatsheets folder in the right place and not inside some unrelated project." />

Start from your home directory.

**macOS / Linux**

```bash
cd ~
mkdir -p cli-academy/cheatsheets
cd cli-academy/cheatsheets
pwd
```

**Windows PowerShell**

```powershell
cd ~
New-Item -ItemType Directory -Path cli-academy\cheatsheets -Force
cd cli-academy\cheatsheets
Get-Location
```

### Step 2 — Create the file and write the header

<StepMeta purpose="Turn the new workspace into a real document with a clear title and opening description." action="Create command-cheat-sheet.md and write the header plus one short description paragraph." expectedResult="A markdown file exists with a title and introductory text instead of an empty file." whyItMatters="Starting with a clean header makes the cheat sheet readable and gives later sections a stable document structure." nextStep="After the header is in place, add the navigation commands you learned in this path." askTutor="I'm on Step 2 of Mini project: create your command cheat sheet. Help me confirm that these commands created the file correctly and explain how to tell whether I overwrote or appended at each line." />

**macOS / Linux**

```bash
echo "# Terminal Command Cheat Sheet" > command-cheat-sheet.md
echo "" >> command-cheat-sheet.md
echo "Commands I learned in CLI Academy Path 02." >> command-cheat-sheet.md
echo "" >> command-cheat-sheet.md
```

**Windows PowerShell**

```powershell
Set-Content command-cheat-sheet.md "# Terminal Command Cheat Sheet"
Add-Content command-cheat-sheet.md ""
Add-Content command-cheat-sheet.md "Commands I learned in CLI Academy Path 02."
Add-Content command-cheat-sheet.md ""
```

### Step 3 — Add the navigation section

<StepMeta purpose="Capture the movement commands from Path 02 while they are still fresh." action="Append the Navigation heading and the listed navigation command bullets to the cheat sheet." expectedResult="The file gains a new navigation section with the key location and movement commands for your platform." whyItMatters="Documenting commands immediately after using them turns short-term memory into a durable reference you can trust later." nextStep="Once navigation is captured, add the commands for creating files and folders." askTutor="I'm on Step 3 of Mini project: create your command cheat sheet. Run each command and then record it in your cheat sheet.

**macOS / Linux**

```bash

echo "- ls — list files and folders" >> command-cheat-sheet.md
```

**Windows PowerShell**


### What can go wrong

| Problem | Cause | Fix |
|---|---|---|
| Used `>` instead of `>>` and wiped the whole file | Single arrow overwrites | Start from Step 2 and rebuild. This time watch for `>>` on every append line |
| File contains weird characters or blank lines | Copy-paste from a website introduced invisible characters | Open the file in VS Code (`code command-cheat-sheet.md`) and clean up manually |
| Quotes appear literally in the file with quotes | Some shells preserve outer quotes | Open in VS Code and remove them, or use single quotes on macOS/Linux |
| `grep: command not found` on Windows | `grep` is a Unix command | Use `Select-String` as shown in the Windows PowerShell command above |
| File ends up in the wrong folder | You forgot to `cd` into `cheatsheets/` | Move it: `mv command-cheat-sheet.md cli-academy/cheatsheets/` or `Move-Item command-cheat-sheet.md cli-academy\cheatsheets\` |
